XML/SWF Charts for CodeIgniter

Documentation

Usage

Using this library involves loading the library, calling the XML related methods on the library (see section "XML Related Methods" below) and then generating the XML.

Example:

//load the library
$this->load->library('maani_xml_swf_charts');

//add XML nodes
$this->maani_xml_swf_charts->chart_type("stacked column");
$this->maani_xml_swf_charts->chart_data(
					array("sales","non-sales"),
					array("Clothing","Food","Toys"),
					array(
						array("100","145"),
						array("34","54"),
						array("84","21")));
$this->maani_xml_swf_charts->chart_label(array(
					"shadow"=>'low',
					"color"=>'ffffff',
					"alpha"=>'95',
					"size"=>'20',
					"position"=>'inside',
					"as_percentage"=>'true'));
	
//generate XML	
$xml_string = $this->maani_xml_swf_charts->generate_xml();


General Methods

generate_xml ([bool $format_xml])

$format_xml: if specified as true, the XML will be formatted in a readable manner. Default is false
returns: a string of the generated XML

clear ()

clears out all the nodes added using the XML related methods

XML Related Methods

chart_border (array $attrs)

$attrs: an associative array of key/value pairs representing the attributes' names/values in the XML/SWF Charts manual

chart_data (array $category_headers, array $value_headers, array $data, [array $attrs])

$category_headers: an array of labels on the independent axis (which is usually the horizontal axis)
$value_headers: an array of labels on the dependent axis (which is usually the vertical axis)
$data: the 2D array of data representing the data to be used to generate the chart
$attrs: a separate array of attributes. Either a single dimensional array of key/value pairs representing attributes to be applied to all data nodes or a 2D array of associative arrays at the same positions of the relevant data in the $data.

The $data array can be in two forms:

  1. containing only data
  2. containing data and individual attributes for each data node. Each element in the 2D array is an associative array with two indices:
    "val": the actual data value
    "attrs": an array of key/value pairs representing the attributes for that data node
    e.g. 
    $data =
	array (
		array (
			array (	
				"val" => 21,
				"attrs" => array (
					"label" => "highest",
					"note" => "busy hour" )),
			array (
				"val" => 7,
				"attrs" => array (
					"label" => "lowest",
					"note" => "lunch"))),
		array (
			array (
				"val" => 18,
				"attrs" => array (
					"label" => "highest",
					"note" => "busy hour" )),
			array (
				"val" => 4,
				"attrs" => array (
					"label" => "lowest",
					"note" => "tea break"))));

Note: do not mix the two kinds of arrays. Either all data nodes must be of type 1 or all must be of type 2 (this issue may be fixed in the future)

draw (array $shapes)

$chart_shapes: a normal (numerically indexed) array whose elements are associative arrays containing the information of each shape to be drawn. These associative arrays of key/value pairs represent the names/values of attributes allowed for each shape in the manual. Two special keys are reserved for the shape type ("_type") and the text for the case of a text shape ("_text"). Note that the "_type" is always required.
e.g.

$shapes = 
	array ( 
		array (
			"_type" => "image",
			"layer" => "background",
			"x" => "0",
			"y" => "17"),
		array (
			"_type" => "text",
			"_text" => "graph",
			"transition"=>"slide_left",
			"delay"=>"1",
			"duration"=">1",
			"x"=>"0"));

filter (array $filters)

$filters: a normal (numerically indexed) array whose elements are associative arrays containing the information of each filter to be used. These associative arrays of key/value pairs represent the names/values of attributes allowed for each shape in the manual. A special keys is reserved for the filter type ("_type"). Note that the "_type" is always required.
e.g.

$filters = 
	array ( 
		array (
			"_type"=>"shadow",
			"id"=>"shadow1",
			"knockout"="true",
			"alpha"=>"75",
			"blurX"="20",
			"blurY"="20"),
		array (
			"_type"=>"bevel",
			"id"=>"bevel2",
			"angle"=>"45",
			"blurX"=>"3",
			"blurY"=>"3",
			"distance"=>"3"
			"highlightAlpha"=>"50"
			"highlightColor"=>"ffffff"));
